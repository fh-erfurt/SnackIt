Installationsanweisung:
1. Installieren Sie XAMPP und aktivieren Sie PHP short tags in der php.ini
2. Öffnen Sie MySQL und importieren Sie zuerst die Snackit_DB.sql aus dem Ordner src/database und anschließend die Snackit_Datensätze.sql
3. Erstellen Sie in C:\xampp\htdocs\ den Ordner "Snackit" und fügen sie den inhalt des "Projekt" Ordners in den Ordner
4. Klicken Sie in XAMPP bei Apache und MySQL auf "Starten"
5. Öffnen Sie einen Browser und geben Sie als URL "http://localhost/Snackit" ein
